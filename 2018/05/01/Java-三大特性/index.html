Java 三大特性

Java的三大特性

今天，我们来了解下Java的三大特性：封装、继承和多态。

封装，顾名思义就是把类的信息隐藏起来，不让外部直接访问，而是得通过该类的方法实现操作和访问。

实现封装的方法：

  1. 修改访问修饰符（改为private）；
  2. 创建setter/getter方法；

  3. 在setter/getter方法里添加属性控制语句。
    例子：

    public class Me{
  private String name;
  public setName(String name){
    this.name=name;
  }
  public getName(){
    return name;
  }
    }

继承，这是类与类之间的一种关系，跟「继承我的花呗」不同意思。比如猫是动物，那么猫类就是继承了动物类，那么猫类就是动物类的子类(也就是派生类)，动物类是猫类的父类(基类)。在Java 中，一个子类只有一个父类，一个父类可以有多个子类。

例子：

//父类
public class Animal{
    public int age;
    public String name;
    public void eat(){
     System.out.println("动物有吃的本领");
    }
}
//子类
public class Cat extends Animal{}

多态是指对象的多种形态，分别是：引用多态和方法多态。继承是实现多态的基础。

引用多态：

父类的引用可以指向本类的对象，也可以指向子类的对象。
例子：

//父类
public class Animal{
   public int age;
   public String name;
   public void eat(){
     System.out.println("动物有吃的本领");
     }
}
//子类
public class Cat extends Animal{}
//测试类
public class Test{
public static void main(String []args){
  //指向本类
  new A1 = new Animal();
  //指向子类
  new A2 = new Cat();
}

}

方法多态：

创建父类对象时，调用父类的方法；
创建子类对象时，调用子类重写的方法或者继承父类的方法。
注意：子类独有的方法父类不能调用。
例子：

//父类
public class Animal{
  public int age;
  public String name;
  public void eat(){
    System.out.println("动物有吃的本领");
    }
}
//子类
public class Cat extends Animal{
  public void eat(){
    System.out.println("猫很能吃");
    }
}
//测试类
public class Test{
  public static void main(String []args){
  new A1=new Animal();
  new A2=new Cat();
  A1.eat();
  A2.eat();
}
}
